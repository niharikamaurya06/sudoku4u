# sudoku4u

handcrafted sudoku game playable in your browser

**[check it out!](https://niharikamaurya06.github.io/sudoku4u)**

---
 
## how to play
 
- click a cell, then click a number or type it on your keyboard
- pre-filled numbers are locked in — only the empty cells are yours to fill
- every row, column, and 3x3 box must contain the digits 1 through 9
- the timer starts on your first move, and errors are counted
- stuck? hit solve

---

## built with
 
- HTML, CSS, vanilla JavaScript
- no libraries, no frameworks, no build step
